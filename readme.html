// 03 — INSTALLATION

Get Running in 5 Steps

01

Clone the Repository

git clone https://github.com/AbdulGhaffarcs/Kryptos-Niarad.git
cd Kryptos-Niarad
02

Install Python Dependencies

pip install streamlit langchain langchain-community langchain-ollama
pip install langchain-core langchain-text-splitters faiss-cpu
pip install pymupdf python-pptx openpyxl docx2txt
03

Pull Ollama Models

ollama pull llama3.2
ollama pull nomic-embed-text
04

Create Theme Config

# Create .streamlit/config.toml
[theme]
base = "dark"
backgroundColor = "#080B0F"
secondaryBackgroundColor = "#0D1117"
textColor = "#C8D6E5"
primaryColor = "#00FFAA"
05

Launch NIARAD

streamlit run app.py

// 05 — PERFORMANCE

Speed Notes

COLD START

First Message

20–60 seconds as Ollama loads the model into RAM. All subsequent messages are much faster.

GREETINGS

Small Talk

1 LLM call only. Router is bypassed entirely for "hi", "hello", and similar phrases.

RAG QUERIES

Document Questions

2 LLM calls: one to classify the query, one to generate the answer from context.
